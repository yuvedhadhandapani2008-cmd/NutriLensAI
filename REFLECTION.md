# Reflection

## What Went Well

The most successful part of the project was integrating Gemini Vision to analyze food images and generate nutrition information. The AI was able to recognize multiple foods, estimate serving sizes, and provide nutritional insights.

Building the React Native UI was also straightforward using Expo and reusable components.

## Challenges Faced

The biggest challenge was integrating image processing and AI responses reliably. During development, I encountered issues with image conversion, Expo FileSystem compatibility, API key management, and Firebase data synchronization.

Generating PDF reports and handling remote image URLs also required several debugging iterations.

## What I Learned

Through this project I learned:

- AI prompt engineering
- Gemini Vision integration
- React Native application architecture
- Firebase Firestore data management
- Analytics and data visualization
- PDF generation in mobile applications
- Environment variable security practices

## What I Would Improve

Given more time, I would add:

- User authentication
- Personalized nutrition recommendations
- Water intake tracking
- Health score analytics
- Cloud backup and synchronization
- Advanced AI coaching features

## Final Thoughts

This project helped me understand how AI can be integrated into real-world mobile applications. Building NutriLens AI improved my skills in frontend development, AI integration, debugging, and product design.
