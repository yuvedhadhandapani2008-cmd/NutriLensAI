# CLAUDE.md

## Project Name

NutriLens AI

## Project Description

NutriLens AI is an AI-powered nutrition tracking mobile application built using React Native and Expo.

The app allows users to:

- upload food images
- capture food photos using camera
- analyze meals using Gemini Vision AI
- receive nutrition insights
- estimate calories and macros
- get fitness recommendations

## Tech Stack

- React Native
- Expo
- JavaScript
- Gemini Vision API
- Expo Camera
- Expo Image Picker

## Current Features

- Food image upload
- Real-time camera capture
- AI food recognition
- Nutrition analysis
- Ingredient breakdown
- Serving estimation
- Dark themed UI
- Result screen navigation
- Meal history
- Daily calorie tracker
- Health score system
- Macro charts
- Personalized recommendations

## Development Notes

- Expo SDK 54 project
- Uses environment variables for API security
- Gemini Vision handles food analysis
- Optimized for Android testing using Expo Go
