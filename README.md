# NutriLens AI 🍎📱

AI-powered nutrition tracking mobile application built with React Native, Expo, Firebase, and Gemini Vision API.

## Overview

NutriLens AI helps users understand their food intake by analyzing food images using artificial intelligence. Users can capture or upload food photos and instantly receive a detailed nutritional breakdown including calories, protein, carbohydrates, fats, and dietary recommendations.

Inspired by modern AI nutrition applications such as Cal AI, NutriLens focuses on delivering fast, accurate, and visually appealing nutrition insights.

## Installation

### Clone Repository

git clone https://github.com/yuvedhadhandapani2008-cmd/NutriLens-AI.git
cd NutriLens-AI

### Install Dependencies

npm install

### Start Project

npx expo start

### Run on Device

- Install Expo Go
- Scan QR code generated by Expo

---

## Environment Variables

Create a `.env` file:

EXPO_PUBLIC_GEMINI_API_KEY=YOUR_API_KEY

Never commit API keys to GitHub.
